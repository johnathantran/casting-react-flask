Endpoint Testing

To test all endpoints, please refer to the Postman Collection found in the root of the submitted project folder.

To test endpoints through the frontend, please run the frontend locally by running 'npm start' in the project root, and then interacting with the React application that launches in the browser.

Note that this API endpoint will return data from the Heroku Postgres database whenever you hit the endpoints through Postman or the React frontend.

Running Locally

Run 'python app.py' in the project root, then run 'npm start'.

To switch to a local database, in the project root, rename the '.env testCopy' to '.env' and give the current '.env' file a different name so it is not overwritten. This will reroute to a local database connection. Please note that you will have to fill out the .env file with your local database info and create the tables Movie and Actor manually.

Finally, to run unit tests on this app, you can run 'test_app.py' in the project root. By default, the tests will run at the local DATABASE_TEST_URL specified in the .env file.
