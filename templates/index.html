Welcome to Casting Agency Application

You have reached the backend server.

For this project, only the backend is deployed on Heroku. You can still access all the data using Postman or a local React frontend app provided.

Endpoint Testing

To test all endpoints, please refer to the Postman Collection found in the root of the submitted project folder.

To test endpoints through the frontend, please run the frontend locally by running 'npm start' in the project root, and then interacting with the React application that launches in the browser.

Note that this API endpoint will return data from the Heroku Postgres database whenever you hit the endpoints through Postman or the React frontend.

Running Locally

Follow the below steps if you want to connect a local database to your frontend (by default, the local frontend will connect to the Heroku database).

  1. 1. Uncomment the local environment variables in '.env' to use, and comment out the Heroku environment variables
  2. 2. Change the 'proxy' key in package.json to value: 'http://localhost:5000/')
  3. 3. Start the local server on port 5000 by running 'python app.py'
  4. 4. Start the frontend with 'npm start'

Please note that you will have to fill out the .env file with your local database info and create the tables Movie and Actor manually.

Finally, to run unit tests on this app, you can run 'test_app.py' in the project root. By default, the tests will run at the local DATABASE_TEST_URL specified in the .env file.
